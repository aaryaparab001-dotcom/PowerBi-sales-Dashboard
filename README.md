# 📊 Power BI Sales Dashboard

## 🔎 Project Overview
This project presents an interactive Power BI dashboard designed to analyze sales performance, customer trends, and revenue insights.

The dashboard helps businesses make data-driven decisions by identifying:
- Sales growth patterns
- Top-performing products
- Regional performance
- Customer behavior insights

## 🛠️ Tools Used
- Power BI
- Excel (Data Cleaning & Preparation)
- Data Visualization Techniques

## 📈 Key Features
- Dynamic filtering using slicers
- KPI indicators for quick insights
- Interactive charts and visuals
- Clean and user-friendly layout

## 📂 Files Included
- `.pbix` file – Main Power BI dashboard
- Dataset used for analysis
- Dashboard screenshots for preview

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it using Microsoft Power BI Desktop
3. Refresh the dataset if needed
4. Explore the interactive dashboard
